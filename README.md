## Использование и функционал

### В данном проекте используются следующие функции:
```
get_mask_card_number()
```
Функция принимает на вход номер карты и возвращает ее маску в формате XXXX XX** **** XXXX, где X — это цифра номера.
```
get_mask_account()
```
Функция принимает на вход номер счета и возвращает его маску в формате **ХХХХ, где ХХХХ последние четыре цифры.
```
mask_account_card()
```
Функция принимает строку, содержащую тип и номер карты или счета и возвращает замаскированный счет или номер карты.
```
get_date()
```
Функция принимает на вход строку с датой в формате "2024-03-11T02:26:18.671407" и возвращает строку с датой в формате 
"ДД.ММ.ГГГГ" ("11.03.2024").
```
deffilter_by_state()
```
Функция принимает список словарей и возвращает новый список словарей, содержащий только те словари, у которых ключ 
state соответствует указанному значению (по умолчанию 'EXECUTED').
```
sort_by_date()
```
Функция принимает список словарей и необязательный параметр, задающий порядок сортировки (по умолчанию — убывание) и 
возвращает новый список отсортированный по дате (date).
```
get_info_about_transactions()
```
Функция принимает путь к файлу с транзакциями и открывает его.
```
get_transaction_amount()
```
Функция принимает транзакцию и возвращает ее сумму. Если в транзакции валюта не является рублями,
то функция конвертирует ее в рубли.

### В данном проекте используются следующие функции-генераторы:
```
filter_by_currency()
```
Функция возвращает итератор, который поочередно выдает транзакции, где валюта операции соответствует 
заданной (например, USD).

Пример использования:  
Необходимо создать переменную (например gen) и указать в нее в качестве аргумента filter_by_currency()  
gen = filter_by_currency(transactions, "USD")  
И далее вызывать поочередно с помощью print(next(gen))
```
transaction_descriptions()
```
Функция-генератор принимает список словарей с транзакциями и возвращает описание каждой операции по очереди.

Пример использования:  
Необходимо создать переменную (например gen) и указать в нее в качестве аргумента transaction_descriptions()  
gen = transaction_descriptions(transactions)  
И далее вызывать поочередно с помощью print(next(gen))
```
card_number_generator()
```
Функция-генератор может сгенерировать номера карт в заданном диапазоне от 0000 0000 0000 0001 до 9999 9999 9999 9999.

Пример использования:  
Необходимо создать переменную (например gen) и указать в нее в качестве аргумента card_number_generator()  
gen = card_number_generator(1, 2) - 1, 5 это диапазон формирования номеров карт  
И далее вызывать поочередно с помощью print(next(gen))  
Пример вывода:  
0000 0000 0000 0001  
0000 0000 0000 0002

### В данном проекте используются следующие декораторы:
```
@log()
```
Декоратор который автоматически логирует начало и конец выполнения функции, а также ее результаты или возникшие ошибки. 
Декоратор способен выводить логи в консоль и в случае передачи необязательного аргумента filename, записывать логи 
в указанный фал.  

Пример использования без filename:  
```
@log()
def my_function(x, y):
    return x + y

print(my_function(1, 2))
``` 
Выведет в консоль:  
my_function ok  
  
3

Пример использования c filename:  
```
@log(filename="mylog.txt")
def my_function(x, y):
    return x + y

print(my_function(1, 2))
``` 
Создаст файл mylog.txt и запишет в него: my_function ok. В данном примере нет цифры 3 так как в файл записываются 
только логи без результатов вычислений.
